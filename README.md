# AD GRAM Bot

Telegram бот для заданий (подписки/просмотры/реакции) с балансом, рефералами,
проверками подписки и оплатой через CryptoPay.

## Быстрый старт

1. Убедитесь, что установлен Python 3.10+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` по примеру `.env.example` и заполните токены.
4. Запустите бота:

```bash
python bot.py
```

## Важно

- Для проверки подписки бот должен быть админом в канале/группе.
- Для спонсоров бот также должен быть админом, чтобы читать статус подписки.
- Для просмотров/реакций рекламодатель пересылает пост боту.
- При создании задания указывается количество выполнений (оплата = награда * количество).
- Токен CryptoPay берется в @CryptoBot → Crypto Pay.
- Для TON оплаты укажите актуальный курс `TON_USD_RATE`.

## Команды

- `/start` — запуск и проверка подписки на спонсоров
- `/admin` — админ-панель (только `ADMIN_IDS`)

## Логи и база

База данных создается автоматически в `bot.db`.
